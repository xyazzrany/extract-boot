# Extract_boot

**这是一个用来提取 boot.img的 Github Action 项目**

## Usage

Fork 本仓库，在 Action 界面内选择 “Extract_boot”然后点击 “Run workflow”，填入 ROM下载直链，等待完成。

## Others
依赖于 [payload-dumper-go](https://github.com/ssut/payload-dumper-go)
